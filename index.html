rhelpi18n

Multilingual R help

Read — and write — R package documentation in your own language.

Reading help in your language →

For R users. Install the engine and a translation, set your language, and see ?function in Spanish, Chinese, Portuguese — whatever's available.

Translator's guide →

For translators. Generate a translation skeleton from a package's source, fill it in (keeping the placeholders), test it locally, and publish it on GitHub.
